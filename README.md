# Программный сервис для предзаказа товаров

Профессиональное решение для автоматизации связки «потребность клиента — ожидаемая поставка». Сервис позволяет бронировать дефицитный товар, находящийся в пути, исключая потерю прибыли из-за человеческого фактора.

---

## Ключевой функционал
*   **Учет транзита:** Регистрация поставок со статусом «В пути».
*   **Автоматическое бронирование:** Списание «виртуального остатка» при создании предзаказа.
*   **Система уведомлений:** Мгновенная рассылка в Telegram при поступлении товара на склад.
*   **Ролевая модель:** Разграничение прав доступа (Администратор, Склад, Клиент).

## Технологический стек
*   **Язык:** Python 3.x.
*   **Backend:** Flask[cite: 17].
*   **БД:** SQLite.
*   **ORM:** SQLAlchemy.
*   **Интеграции:** Telegram Bot API[cite: 17, 19].
