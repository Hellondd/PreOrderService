# PreOrderService PROTO

## Что умеет этот прототип:

    БД: Сохраняет все данные в реальный файл. При перезагрузке ничего не пропадает.

    Логика: Не дает забронировать больше товара, чем едет в поставках.

    Интеграция: При нажатии кнопки "Принять на склад", система автоматически пробегает по очереди предзаказов и меняет статус на "Ready" тем, кому хватило товара из этой партии.
