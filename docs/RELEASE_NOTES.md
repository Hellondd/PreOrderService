# Release Notes

## Версия 1.0.0 (Production Build)
**Дата:** 07.06.2026

**Что вошло в версию:**
- Модуль `orders.py` для динамического расчета остатков.
- Интеграция с Telegram API (`notifications.py`) без хардкода токенов.
- Настроен Gunicorn для обработки запросов в production.
- Развертывание в облачной инфраструктуре PaaS.

**Известные ограничения:**
- База данных SQLite используется в режиме эфемерного хранилища. Данные обнуляются при рестарте контейнера (допустимо для демо-стенда).
