# BACKUP_RESTORE_REPORT.md

## Тип хранилища
SQLite (файл `warehouse.db` в корне проекта)

## Процедура бэкапа
1. Создана папка `backups/`.
2. Скрипт `scripts/backup.bat` копирует `warehouse.db` в `backups/backup_YYYYMMDD_HHMMSS.db`.

## Тестирование восстановления
1. Остановлено приложение.
2. Переименован `warehouse.db` в `warehouse.db.old`.
3. Запущен `scripts/restore.bat`, выбран последний бэкап.
4. Файл скопирован обратно в `warehouse.db`.
5. Запущено приложение, выполнена проверка данных (заказы, пользователи).

## Результат
- ✅ Бэкап создаётся успешно.
- ✅ Восстановление работает, данные не теряются.
- ⚠️ Ручной режим – рекомендована автоматизация.
